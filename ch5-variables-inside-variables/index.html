5.5 arrays in objects

A variable can contain an array as its value. A property of an object can also contain an array as its value. Suppose the four spells are the spells Luna knows. Add an array of spells to the "spells" property of luna.

If you are succesfull, the following code would work to access the first spell: console.log(wizard.spells[0]), where wizard is the name of the variable containing the object.

5.6 objects in arrays

A variable can contain an object as its value. An array can also contain multiple objects as its items. Make a "wizards" array containing not only Luna (and all her properties), but also another wizard (and all of his/her properties).

If you are succesfull, the following code would work to access the second wizards name: console.log(wizards[1].name).
